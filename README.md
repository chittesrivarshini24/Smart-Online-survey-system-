# SMART ONLINE SURVEY SYSTEM

## Introduction

Smart Online Survey System is a web application used to collect feedback from Students, Hospitals, Employees, and Offices. The system allows users to register, log in, submit survey responses, and view analytics.

## Objectives

* Collect feedback online.
* Store responses securely.
* Display survey statistics.
* Provide easy-to-use survey forms.

## Technologies Used

* HTML
* CSS
* JavaScript
* Firebase Authentication
* Firebase Firestore
* Chart.js

## Modules

1. Registration
2. Login
3. Dashboard
4. Student Survey
5. Hospital Survey
6. Employee Survey
7. Office Survey
8. Analytics Dashboard

## Features

* User Authentication
* Survey Submission
* Firebase Database Storage
* Pie Chart Analytics
* Bar Chart Analytics
* Responsive Design

## Conclusion

The Smart Online Survey System provides an efficient way to collect and analyze feedback. It helps organizations make better decisions using survey data and graphical reports.
